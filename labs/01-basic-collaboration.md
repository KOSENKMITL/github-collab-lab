# Lab 01 — Basic GitHub Collaboration

## Objectives

By the end of this lab, you should be able to:

- Work with a GitHub repository locally (clone, branch, commit, push)
- Edit a file and add your information
- Create a Pull Request (PR) to propose changes

---

## Prerequisites

- Git installed on your machine
- GitHub account
- Access to the Classroom assignment repository

---

## Part 1 — Clone Your Repository

1. Go to your assignment repository page on GitHub.
2. Click **Code** → copy the **HTTPS** or **SSH** URL.
3. On your computer, run:

   ```bash
   git clone <your-repo-url>
   cd <your-repo-folder>
